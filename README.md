# TWITTX
This program will scrape all tweet from twitter in your current coordinate and on the specific radius.

# INSTALATION
```console
git clone https://github.com/InersIn/twittx.git
```

```console
cd twittx
```

```console
pip3 install -r ./requirements.txt
```

# RUN PROGRAM
```console
python3 twittx.py
```
